<h1>Simple-Cli-News</h1>

### Prerequisites
Just make sure you have cargo installed
```sh
curl https://sh.rustup.rs -sSf | sh
```

### Installation

```sh
cargo install simple-cli-news
```

<!-- USAGE EXAMPLES -->
## Usage

Print top headlines:
```sh 
simple-cli-news
```
Print news including a query:
```sh
simple-cli-news q <query>
```
